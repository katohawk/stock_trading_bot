# OKX 交易机器人 Web 控制台

暗黑交易风界面：启动/暂停、参数配置、实时日志、失败自动重试。

## 启动

```bash
cd /Users/hekun/Downloads/stock_trading_bot
pip install -r requirements.txt   # 含 flask
python server.py --port 5555
```

浏览器打开: **http://127.0.0.1:5555/**

## 功能

- **启动 / 暂停**：控制机器人子进程，无需再在命令行执行 `run_okx_live.py`。
- **参数**：在页面填写交易对、触发比例、轮询间隔、冷静期、买入金额、是否实盘/模拟盘等，启动时生效。
- **实时日志**：页面底部通过 SSE 实时输出机器人 stdout，与命令行一致。
- **自动重试**：进程异常退出（非手动暂停）时，默认 10 秒后自动重试，最多 5 次。

## 参数说明

| 参数 | 说明 |
|------|------|
| 交易对 | 如 BTC/USDT |
| 触发比例 % | 涨跌超过该比例触发买卖，默认 0.5 |
| 轮询间隔(秒) | 每次检查间隔，默认 60 |
| 冷静期(秒) | 同币种两次下单最小间隔 |
| 每次买入 USDT | 单次买入金额 |
| 最小下单 USDT | 低于此不下单 |
| 实盘下单 | 勾选后真实下单（需 .env 配置 OKX API） |
| 模拟盘 | 勾选后走 OKX 模拟盘 |

## 部署

- 本机使用：`python server.py` 即可，默认 `http://127.0.0.1:5555`。
- 局域网访问：`python server.py --host 0.0.0.0 --port 5555`，然后用 `http://<本机IP>:5555/` 访问。
- 生产环境建议用 gunicorn：`gunicorn -w 1 -b 0.0.0.0:5555 server:app`。
